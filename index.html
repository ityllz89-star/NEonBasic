About

NeonBasic is a tiny BASIC‑style scripting language concept inspired by MiniP. This page is for learning and documentation purposes, and you can keep expanding the language.

Usage: Write code in the Live Demo below and press "Run".

Basic Commands

  • PRINT "text" — outputs text.
  • VAR <name> <number> — defines a variable (numbers only).
  • IF <var> > <number> THEN PRINT "text" — simple condition (>, <, == supported).
  • REM ... — comment line.

Quick Example

REM Example program
VAR A 8
IF A > 5 THEN PRINT "A is greater than 5"
PRINT "Done!"
